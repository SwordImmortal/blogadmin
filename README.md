# blogadmin
项目中包含是平时自己整理的常用代码，其中还包括一个简单的博客后台管理系统，和java开发小工具，后台管理系统正在开发中，小工具包含的功能如下：
### 基础工具
1. 大小写转换
2. 提取代码中的sql
3. 用StringBuilder拼接sql
4. 提取实体类的set方法
5. js格式化
6. html格式化
7. sql格式化

### 实体类生成 （需要连接数据库）
1. 生成实体类，可以自定义数据源，是否hibernate注释，是否字段注释，是否继承基础类
2. 获取相关增删改查相关sql
3. 获取springjdbc Mapper映射类

### 拼接sql
1. hibernate 或者 springjdbc sql 拼接
